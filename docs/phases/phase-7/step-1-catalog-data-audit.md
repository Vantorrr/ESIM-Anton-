# Шаг 1. Провести catalog data audit

## Что нужно сделать

- проверить `tags`, `notes`, `region`, `supportTopup`;
- сопоставить спорные тарифы из клиентского списка.

## Результат шага

Есть список тарифов, которым нужны пометки или правки данных.
